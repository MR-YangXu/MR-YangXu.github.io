# 【Q019】匿名函数

Issue

欢迎在 Gtihub Issue 中回答此问题: Issue 19open in new window

Author

回答者: kangyanaopen in new window

# 1. 什么是匿名函数

  • 匿名函数，即没有名称的函数
  • 如果单独只写一个匿名函数，此时是不符合语法要求的会报错。需要给 匿名函数 包裹一个括号，使之成为 表达式
  • 被小括号包裹的内容会被js识别为一个函数表达式

# 2. 创建方式

(function () {
      // to something...
}());
